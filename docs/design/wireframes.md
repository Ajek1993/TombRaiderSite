## Design Notes & Specifications

### Grid System
- **Desktop:** 12-column grid
- **Tablet:** 8-column grid
- **Mobile:** 4-column grid

### Spacing
- **Section padding:** 80px (desktop), 40px (mobile)
- **Card margin:** 24px
- **Element spacing:** 16px, 24px, 32px increments

### Breakpoints
```css
/* Mobile First */
@media (min-width: 768px) { /* Tablet */ }
@media (min-width: 1024px) { /* Desktop */ }
@media (min-width: 1440px) { /* Large Desktop */ }
```

### Card Dimensions
- **Gameplay card:** 320px Ã— 240px (thumbnail)
- **Highlight card:** 280px Ã— 180px
- **Featured card:** 640px Ã— 360px

### Interactive Elements
- **Hover elevation:** translateY(-4px)
- **Transition duration:** 0.3s ease
- **Neon glow on hover:** box-shadow expansion

### Navigation
- **Sticky navigation:** position: fixed, top: 0
- **Height:** 80px (desktop), 60px (mobile)
- **Z-index:** 1000

### Typography Hierarchy
- **H1:** 48px (desktop), 32px (mobile)
- **H2:** 36px (desktop), 28px (mobile)
- **H3:** 24px (desktop), 20px (mobile)
- **Body:** 16px
- **Small:** 14px

## Accessibility Features

### Color Contrast
- All text meets WCAG AA standards
- Interactive elements have clear focus states
- Alternative color schemes for colorblind users

### Keyboard Navigation
- Tab order follows visual hierarchy
- Focus indicators visible
- Skip to content link

### Screen Readers
- Semantic HTML5 elements
- ARIA labels for icons
- Alt text for all images

- Multiple sizes served based on viewport
- Lazy loading for below-fold content
- Placeholder while loading
